Kharada Language Translation System

An AI-powered translation system developed to translate the Kharada language into English using deep learning techniques.

🎯 Objective

To preserve the Kharada language and provide accurate English translations using advanced Natural Language Processing (NLP) and Deep Learning models.

💻 Technologies Used

  • Python
  • Flask
  • PyTorch
  • Transformer Model
  • LSTM Model
  • SentencePiece
  • SQLite Database
  • HTML, CSS & JavaScript

🤖 Models Used

  • Transformer Model
  • LSTM Model

🚀 Future Scope

  • Speech-to-Text Translation
  • Voice Assistant Integration
  • Android & iOS Mobile Application
  • Multilingual Translation
  • Cloud Deployment
  • Larger Kharada Dataset
