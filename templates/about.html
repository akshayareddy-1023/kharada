About Kharada Translator

AI Based Kharada Language Translation System

About The Project

Kharada Translator is an AI-powered language translation system developed to convert Kharada language text into English language accurately. The system uses Natural Language Processing (NLP) techniques and transformer-based models to understand language patterns and generate meaningful translations.


Key Features

Language Translation

Converts Kharada text into English.

Voice Input

Supports speech based input.

Translation History

Stores previous translations.



Technologies Used

  • Python
  • Flask Framework
  • HTML, CSS, JavaScript
  • Natural Language Processing (NLP)
  • Transformer Based Translation Model
